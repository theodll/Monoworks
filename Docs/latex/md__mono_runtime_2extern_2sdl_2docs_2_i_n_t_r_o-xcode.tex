The easiest way to use SDL is to include it as a subproject in your project.

We\textquotesingle{}ll start by creating a simple project to build and run \href{hello.c}{\texttt{hello.\+c}}


\begin{DoxyItemize}
\item Create a new project in Xcode, using the App template and selecting Objective C as the language
\item Remove the .h and .m files that were automatically added to the project
\item Remove the main storyboard that was automatically added to the project
\item On i\+OS projects, select the project, select the main target, select the Info tab, look for "{}\+Custom i\+OS Target Properties"{}, and remove "{}\+Main storyboard base file name"{} and "{}\+Application Scene Manifest"{}
\item Right click the project and select "{}\+Add Files to \+[project]\+"{}, navigate to the SDL docs directory and add the file hello.\+c
\item Right click the project and select "{}\+Add Files to \+[project]\+"{}, navigate to the SDL Xcode/\+\+SDL directory and add SDL.\+xcodeproj
\item Select the project, select the main target, select the General tab, look for "{}\+Frameworks, Libraries, and Embedded Content"{}, and add SDL3.\+framework
\item Build and run! 
\end{DoxyItemize}
