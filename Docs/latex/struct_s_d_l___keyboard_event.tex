\doxysubsection{Detailed Description}
Keyboard button event structure (event.\+key.\texorpdfstring{$\ast$}{*})

The {\ttfamily \doxylink{struct_s_d_l___keyboard_event_a4f16d759dd942640caf2eac761395ec9}{key}} is the base SDL\+\_\+\+Keycode generated by pressing the {\ttfamily \doxylink{struct_s_d_l___keyboard_event_aa3da19b3d3012b50671b7da43b94269f}{scancode}} using the current keyboard layout, applying any options specified in SDL\+\_\+\+HINT\+\_\+\+KEYCODE\+\_\+\+OPTIONS. You can get the SDL\+\_\+\+Keycode corresponding to the event scancode and modifiers directly from the keyboard layout, bypassing SDL\+\_\+\+HINT\+\_\+\+KEYCODE\+\_\+\+OPTIONS, by calling SDL\+\_\+\+Get\+Key\+From\+Scancode().
