\doxysubsection{Detailed Description}
A structure specifying the parameters of an indexed indirect draw command.

Note that the {\ttfamily first\+\_\+vertex} and {\ttfamily \doxylink{struct_s_d_l___g_p_u_indexed_indirect_draw_command_a8f9751b0dd3d82e909ecd70038afe4fc}{first\+\_\+instance}} parameters are NOT compatible with built-\/in vertex/\+instance ID variables in shaders (for example, SV\+\_\+\+Vertex\+ID); GPU APIs and shader languages do not define these built-\/in variables consistently, so if your shader depends on them, the only way to keep behavior consistent and portable is to always pass 0 for the correlating parameter in the draw calls.
