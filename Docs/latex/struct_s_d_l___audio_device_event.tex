\doxysubsection{Detailed Description}
Audio device event structure (event.\+adevice.\texorpdfstring{$\ast$}{*})

Note that SDL will send a SDL\+\_\+\+EVENT\+\_\+\+AUDIO\+\_\+\+DEVICE\+\_\+\+ADDED event for every device it discovers during initialization. After that, this event will only arrive when a device is hotplugged during the program\textquotesingle{}s run.
