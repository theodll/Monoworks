\doxysubsection{Detailed Description}
A structure specifying additional options when using Vulkan.

When no such structure is provided, SDL will use Vulkan API version 1.\+0 and a minimal set of features. The requested API version influences how the feature\+\_\+list is processed by SDL. When requesting API version 1.\+0, the feature\+\_\+list is ignored. Only the vulkan\+\_\+10\+\_\+physical\+\_\+device\+\_\+features and the extension lists are used. When requesting API version 1.\+1, the feature\+\_\+list is scanned for feature structures introduced in Vulkan 1.\+1. When requesting Vulkan 1.\+2 or higher, the feature\+\_\+list is additionally scanned for compound feature structs such as \doxylink{struct_vk_physical_device_vulkan11_features}{Vk\+Physical\+Device\+Vulkan11\+Features}. The device and instance extension lists, as well as vulkan\+\_\+10\+\_\+physical\+\_\+device\+\_\+features, are always processed.
