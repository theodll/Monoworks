\doxysubsection{Detailed Description}
Gamepad device event structure (event.\+gdevice.\texorpdfstring{$\ast$}{*})

Joysticks that are supported gamepads receive both an \doxylink{struct_s_d_l___joy_device_event}{SDL\+\_\+\+Joy\+Device\+Event} and an \doxylink{struct_s_d_l___gamepad_device_event}{SDL\+\_\+\+Gamepad\+Device\+Event}.

SDL will send GAMEPAD\+\_\+\+ADDED events for joysticks that are already plugged in during SDL\+\_\+\+Init() and are recognized as gamepads. It will also send events for joysticks that get gamepad mappings at runtime.
