Demonstrates that unpatched Tracy fails to show GPU zones for kernels launched via CUDA Graphs ({\ttfamily cuda\+Graph\+Launch}).\hypertarget{dir_ab80ffe10545e11618b94653a389c000_autotoc_md1144}{}\doxysubsection{\texorpdfstring{Root cause}{Root cause}}\label{dir_ab80ffe10545e11618b94653a389c000_autotoc_md1144}
When kernels are launched through CUDA Graphs, CUPTI delivers {\ttfamily CONCURRENT\+\_\+\+KERNEL} and {\ttfamily MEMCPY} activity records but no corresponding API callback fires for the individual kernel launches. Tracy\textquotesingle{}s {\ttfamily match\+Activity\+To\+APICall()} always fails, and {\ttfamily match\+Error()} silently drops every GPU zone.\hypertarget{dir_ab80ffe10545e11618b94653a389c000_autotoc_md1145}{}\doxysubsection{\texorpdfstring{Build and run}{Build and run}}\label{dir_ab80ffe10545e11618b94653a389c000_autotoc_md1145}
