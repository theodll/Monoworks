SDL is fully supported on BSD platforms, and is built using \doxysectlink{md__mono_runtime_2extern_2sdl_2docs_2_r_e_a_d_m_e-cmake}{CMake}{0}.

If you want to run on the console, you can take a look at \doxysectlink{md__mono_runtime_2extern_2sdl_2docs_2_r_e_a_d_m_e-kmsbsd}{KMSDRM support on BSD}{0}

SDL is \href{README-platforms.md\#setuid}{\texttt{not designed to be used in setuid or setgid executables}}.
