\begin{DoxyItemize}
\item Currently only software rendering is supported.
\item SDL3\+\_\+main should be used to ensure ROMFS is enabled -\/ this is done with {\ttfamily \#include \texorpdfstring{$<$}{<}\doxylink{_s_d_l__main_8h_source}{SDL3/\+\+SDL\+\_\+main.\+h}\texorpdfstring{$>$}{>}} in the source file that contains your main function.
\item By default, the extra L2 cache and higher clock speeds of the New 2/\+3\+DS lineup are enabled. If you wish to turn it off, use {\ttfamily os\+Set\+Speedup\+Enable(false)} in your main function.
\item {\ttfamily SDL\+\_\+\+Get\+Base\+Path} returns the romfs root instead of the executable\textquotesingle{}s directory.
\item The Nintendo 3DS uses a cooperative threading model on a single core, meaning a thread will never yield unless done manually through the {\ttfamily SDL\+\_\+\+Delay} functions, or blocking waits ({\ttfamily SDL\+\_\+\+Lock\+Mutex}, {\ttfamily SDL\+\_\+\+Wait\+Semaphore}, {\ttfamily SDL\+\_\+\+Wait\+Condition}, {\ttfamily SDL\+\_\+\+Wait\+Thread}). To avoid starving other threads, {\ttfamily SDL\+\_\+\+Try\+Wait\+Semaphore} and {\ttfamily SDL\+\_\+\+Wait\+Semaphore\+Timeout} will yield if they fail to acquire the semaphore, see \href{https://github.com/libsdl-org/SDL/pull/6776}{\texttt{https:\+/\+/\+github.\+com/\+libsdl-\/org/\+\+SDL/\+pull/\+6776}} for more information. 
\end{DoxyItemize}
