\doxysubsection{Detailed Description}
Keyboard text input event structure (event.\+text.\texorpdfstring{$\ast$}{*})

This event will never be delivered unless text input is enabled by calling SDL\+\_\+\+Start\+Text\+Input(). Text input is disabled by default!
