\doxysubsection{Detailed Description}
Pressure-\/sensitive pen motion event structure (event.\+pmotion.\texorpdfstring{$\ast$}{*})

Depending on the hardware, you may get motion events when the pen is not touching a tablet, for tracking a pen even when it isn\textquotesingle{}t drawing. You should listen for SDL\+\_\+\+EVENT\+\_\+\+PEN\+\_\+\+DOWN and SDL\+\_\+\+EVENT\+\_\+\+PEN\+\_\+\+UP events, or check {\ttfamily \doxylink{struct_s_d_l___pen_motion_event_af47731c8a688710d2ac4e306f528cc4e}{pen\+\_\+state} \& SDL\+\_\+\+PEN\+\_\+\+INPUT\+\_\+\+DOWN} to decide if a pen is "{}drawing"{} when dealing with pen motion.
