Performs RQ factorisation of a matrix. Returns 2 matrices, r and q, such that r is an upper triangular matrix, the rows of q are orthonormal and span the same subspace than those of the input matrix, and r\texorpdfstring{$\ast$}{*}q=in. Note that in the context of RQ factorisation, the diagonal is seen as starting in the lower-\/right corner of the matrix, instead of the usual upper-\/left. Given an n-\/by-\/m input matrix, r has dimensions min(n,m)-\/by-\/m, and q has dimensions n-\/by-\/min(n,m).
