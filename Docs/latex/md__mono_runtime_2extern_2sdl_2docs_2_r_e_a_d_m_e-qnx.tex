\chapter{QNX }
\hypertarget{md__mono_runtime_2extern_2sdl_2docs_2_r_e_a_d_m_e-qnx}{}\label{md__mono_runtime_2extern_2sdl_2docs_2_r_e_a_d_m_e-qnx}\index{QNX@{QNX}}
SDL port for QNX, providing both screen and Wayland video backends.

This was originally contributed by Elad Lahav for QNX 7.\+0.

The port was later improved and adapted for QNX 8.\+0 by:\+
\begin{DoxyItemize}
\item Ethan Leir
\item Roberto Speranza
\item Darcy Phipps
\item Jai Moraes
\item Pierce Mc\+Kinnon
\end{DoxyItemize}

Further changes to enable Wayland with the EGL backend were made by Felix Xing and Aaron Bassett.\hypertarget{md__mono_runtime_2extern_2sdl_2docs_2_r_e_a_d_m_e-qnx_autotoc_md1441}{}\doxysection{\texorpdfstring{Building}{Building}}\label{md__mono_runtime_2extern_2sdl_2docs_2_r_e_a_d_m_e-qnx_autotoc_md1441}
