\subsubsection*{template$<$typename T, size\+\_\+t SIZE = inline\+\_\+buffer\+\_\+size, typename Allocator = detail::\+allocator$<$\+T$>$$>$\newline
class basic\+\_\+memory\+\_\+buffer$<$ T, SIZE, Allocator $>$}
A dynamically growing memory buffer for trivially copyable/\+constructible types with the first {\ttfamily SIZE} elements stored in the object itself. Most commonly used via the {\ttfamily memory\+\_\+buffer} alias for {\ttfamily char}.

{\bfseries{Example}}:\+ \begin{DoxyVerb}auto out = fmt::memory_buffer();
fmt::format_to(std::back_inserter(out), "The answer is {}.", 42);
\end{DoxyVerb}


This will append "{}\+The answer is 42."{} to {\ttfamily out}. The buffer content can be converted to {\ttfamily std::\+string} with {\ttfamily to\+\_\+string(out)}.
