Tracy has a variety of settings that can be adjusted to suit your needs. These settings can be found by clicking on the {\itshape  Wrench} icon on the welcome screen. This will open the about window, where you can expand the {\itshape  Global settings} menu.

The settings are saved between sessions, so you only need to set them once.
