The easiest way to use SDL is to include it as a subproject in your project.

We\textquotesingle{}ll start by creating a simple project to build and run \href{hello.c}{\texttt{hello.\+c}}


\begin{DoxyItemize}
\item Get a copy of the SDL source, you can clone the repo, or download the "{}\+Source Code"{} asset from \href{https://github.com/libsdl-org/SDL/releases/latest}{\texttt{the latest release.}}
\begin{DoxyItemize}
\item If you\textquotesingle{}ve downloaded a release, make sure to extract the contents somewhere you can find it.
\end{DoxyItemize}
\item Create a new project in Visual Studio, using the C++ Empty Project template
\item Add hello.\+c to the Source Files
\item Right click the solution, select add an existing project, navigate to {\ttfamily Visual\+C/\+\+SDL} from within the source you cloned or downloaded above and add SDL.\+vcxproj
\item Select your main project and go to Project -\/\texorpdfstring{$>$}{>} Add -\/\texorpdfstring{$>$}{>} Reference and select SDL3
\item Select your main project and go to Project -\/\texorpdfstring{$>$}{>} Properties, set the filter at the top to "{}\+All Configurations"{} and "{}\+All Platforms"{}, select C/\+\+C++ -\/\texorpdfstring{$>$}{>} General and add the SDL include directory to "{}\+Additional Include Directories"{}
\item Build and run! 
\end{DoxyItemize}
