\begin{DoxyItemize}
\item gles1/\+gles2 support and renderers are disabled by default and can be enabled by configuring with {\ttfamily -\/DVIDEO\+\_\+\+VITA\+\_\+\+PVR=ON} These renderers support 720p and 1080i resolutions. These can be specified with:\+ {\ttfamily SDL\+\_\+\+Set\+Hint(SDL\+\_\+\+HINT\+\_\+\+VITA\+\_\+\+RESOLUTION, "{}720"{});} and {\ttfamily SDL\+\_\+\+Set\+Hint(SDL\+\_\+\+HINT\+\_\+\+VITA\+\_\+\+RESOLUTION, "{}1080"{});}
\item Desktop GL 1.\+X and 2.\+X support and renderers are also disabled by default and also can be enabled with {\ttfamily -\/DVIDEO\+\_\+\+VITA\+\_\+\+PVR=ON} as long as gl4es4vita is present in your SDK. They support the same resolutions as the gles1/\+gles2 backends and require specifying {\ttfamily SDL\+\_\+\+Set\+Hint(SDL\+\_\+\+HINT\+\_\+\+VITA\+\_\+\+PVR\+\_\+\+OPENGL, "{}1"{});} anytime before video subsystem initialization.
\item gles2 support via PIB is disabled by default and can be enabled by configuring with {\ttfamily -\/DVIDEO\+\_\+\+VITA\+\_\+\+PIB=ON}
\item By default SDL emits mouse events for touch events on every touchscreen. Vita has two touchscreens, so it\textquotesingle{}s recommended to use {\ttfamily SDL\+\_\+\+Set\+Hint(SDL\+\_\+\+HINT\+\_\+\+TOUCH\+\_\+\+MOUSE\+\_\+\+EVENTS, "{}0"{});} and handle touch events instead. Individual touchscreens can be disabled with:\+ {\ttfamily SDL\+\_\+\+Set\+Hint(SDL\+\_\+\+HINT\+\_\+\+VITA\+\_\+\+ENABLE\+\_\+\+FRONT\+\_\+\+TOUCH, "{}0"{});} and {\ttfamily SDL\+\_\+\+Set\+Hint(SDL\+\_\+\+HINT\+\_\+\+VITA\+\_\+\+ENABLE\+\_\+\+BACK\+\_\+\+TOUCH, "{}0"{});}
\item Support for L2/\+\+R2/\+\+R3/\+\+R3 buttons, haptic feedback and gamepad led only available on PSTV, or when using external ds4 gamepad on vita. 
\end{DoxyItemize}
