We\textquotesingle{}ll start by creating a simple project to build and run \href{hello.c}{\texttt{hello.\+c}}


\begin{DoxyItemize}
\item Use our handy script to create a template project:\+ 
\begin{DoxyCode}{0}
\DoxyCodeLine{./build-\/scripts/create-\/android-\/project.py\ org.libsdl.hello\ docs/hello.c}

\item Run Android Studio and open the newly created build/\+org.\+libsdl.\+hello directory
\item Build and run!
\end{DoxyItemize}
