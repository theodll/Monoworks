\doxysubsection{Detailed Description}
A structure specifying the parameters of an indirect draw command.

Note that the {\ttfamily \doxylink{struct_s_d_l___g_p_u_indirect_draw_command_a74726c16ed151dc1121b0ecc5601ad06}{first\+\_\+vertex}} and {\ttfamily \doxylink{struct_s_d_l___g_p_u_indirect_draw_command_a5a50c09f0c1828f0f658ac8385e0d429}{first\+\_\+instance}} parameters are NOT compatible with built-\/in vertex/\+instance ID variables in shaders (for example, SV\+\_\+\+Vertex\+ID); GPU APIs and shader languages do not define these built-\/in variables consistently, so if your shader depends on them, the only way to keep behavior consistent and portable is to always pass 0 for the correlating parameter in the draw calls.
