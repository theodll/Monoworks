\doxysubsubsection{Detailed Description}
Include \texorpdfstring{$<$}{<}\doxylink{pca_8hpp}{glm/\+gtx/\+pca.\+hpp}\texorpdfstring{$>$}{>} to use the features of this extension.

Implements functions required for fundamental \textquotesingle{}princple component analysis\textquotesingle{} in 2D, 3D, and 4D:\+ 1) Computing a covariance matrics from a list of {\itshape relative} position vectors 2) Compute the eigenvalues and eigenvectors of the covariance matrics This is useful, e.\+g., to compute an object-\/aligned bounding box from vertices of an object. \href{https://en.wikipedia.org/wiki/Principal_component_analysis}{\texttt{https:\+/\+/\+en.\+wikipedia.\+org/\+wiki/\+\+Principal\+\_\+component\+\_\+analysis}}

Assuming the provided covariance matrix {\ttfamily covar\+Mat} is symmetric and real-\/valued, this function find the {\ttfamily D} Eigenvalues of the matrix, and also provides the corresponding Eigenvectors. Note:\+ the data in {\ttfamily out\+Eigenvalues} and {\ttfamily out\+Eigenvectors} are in matching order, i.\+e. {\ttfamily out\+Eigenvector\+[i]\+} is the Eigenvector of the Eigenvalue {\ttfamily out\+Eigenvalue\+[i]\+}. This is a numeric implementation to find the Eigenvalues, using \textquotesingle{}QL decomposition\`{} (variant of QR decomposition:\+ \href{https://en.wikipedia.org/wiki/QR_decomposition}{\texttt{https:\+/\+/\+en.\+wikipedia.\+org/\+wiki/\+\+QR\+\_\+decomposition}}).
