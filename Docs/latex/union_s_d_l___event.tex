\doxysubsection{Detailed Description}
The structure for all events in SDL.

The \doxylink{union_s_d_l___event}{SDL\+\_\+\+Event} structure is the core of all event handling in SDL. \doxylink{union_s_d_l___event}{SDL\+\_\+\+Event} is a union of all event structures used in SDL.

This struct is available since SDL 3.\+2.\+0.
