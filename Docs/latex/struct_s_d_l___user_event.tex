\doxysubsection{Detailed Description}
A user-\/defined event type (event.\+user.\texorpdfstring{$\ast$}{*})

This event is unique; it is never created by SDL, but only by the application. The event can be pushed onto the event queue using SDL\+\_\+\+Push\+Event(). The contents of the structure members are completely up to the programmer; the only requirement is that \textquotesingle{}\textquotesingle{}\textquotesingle{}type\textquotesingle{}\textquotesingle{}\textquotesingle{} is a value obtained from SDL\+\_\+\+Register\+Events().
