\doxysubsection{Detailed Description}
A structure specifying parameters related to transferring data to or from a texture.

If either of {\ttfamily \doxylink{struct_s_d_l___g_p_u_texture_transfer_info_a88bd2cc67ced0dcb8b286af477887179}{pixels\+\_\+per\+\_\+row}} or {\ttfamily \doxylink{struct_s_d_l___g_p_u_texture_transfer_info_ae57e799b9fc5487fdd4b2000667bc620}{rows\+\_\+per\+\_\+layer}} is zero, then width and height of passed \doxylink{struct_s_d_l___g_p_u_texture_region}{SDL\+\_\+\+GPUTexture\+Region} to SDL\+\_\+\+Upload\+To\+GPUTexture or SDL\+\_\+\+Download\+From\+GPUTexture are used as default values respectively and data is considered to be tightly packed.

{\bfseries{WARNING}}:\+ On some older/\+integrated hardware, Direct3D 12 requires texture data row pitch to be 256 byte aligned, and offsets to be aligned to 512 bytes. If they are not, SDL will make a temporary copy of the data that is properly aligned, but this adds overhead to the transfer process. Apps can avoid this by aligning their data appropriately, or using a different GPU backend than Direct3D 12.
