\doxysubsection{Detailed Description}
!\+[glm]\+(/\+doc/\+manual/\+logo-\/mini.png)

\href{http://glm.g-truc.net/}{\texttt{Open\+GL Mathematics}} ({\itshape GLM}) is a header only C++ mathematics library for graphics software based on the \href{https://registry.khronos.org/OpenGL/specs/gl/GLSLangSpec.4.60.pdf}{\texttt{Open\+GL Shading Language (GLSL) specifications}}.

{\itshape GLM} provides classes and functions designed and implemented with the same naming conventions and functionality than {\itshape GLSL} so that anyone who knows {\itshape GLSL}, can use {\itshape GLM} as well in C++.

This project isn\textquotesingle{}t limited to {\itshape GLSL} features. An extension system, based on the {\itshape GLSL} extension conventions, provides extended capabilities:\+ matrix transformations, quaternions, data packing, random numbers, noise, etc...

This library works perfectly with {\itshape \href{https://www.opengl.org}{\texttt{Open\+GL}}} but it also ensures interoperability with other third party libraries and SDK. It is a good candidate for software rendering (raytracing /\+ rasterisation), image processing, physics simulations and any development context that requires a simple and convenient mathematics library.

{\itshape GLM} requires a C++ 17 compiler. For older C++ support, GLM 1.\+0 branch can be used. It is a platform independent library with no dependence and it officially supports the following compilers:\+
\begin{DoxyItemize}
\item \href{http://gcc.gnu.org/}{\texttt{{\itshape GCC}}} 8 and higher
\item \href{http://llvm.org/}{\texttt{{\itshape Clang}}} 6 and higher
\item \href{https://developer.apple.com/library/mac/documentation/CompilerTools/Conceptual/LLVMCompilerOverview/index.html}{\texttt{{\itshape Apple Clang 6.\+0}}} and higher
\item \href{http://www.visualstudio.com/}{\texttt{{\itshape Visual C++}}} 2019 and higher
\item \href{https://developer.nvidia.com/about-cuda}{\texttt{{\itshape CUDA}}} 9.\+0 and higher (experimental)
\item Any C++17 compiler
\end{DoxyItemize}

For more information about {\itshape GLM}, please have a look at the \doxysectlink{md__mono_engine_2thirdparty_2glm_2manual}{manual}{0} and the \href{http://glm.g-truc.net/0.9.9/api/modules.html}{\texttt{API reference documentation}}. The source code and the documentation are licensed under either the \href{manual.md\#section0}{\texttt{Happy Bunny License (Modified MIT) or the MIT License}}.
