Let\textquotesingle{}s start our adventure by instrumenting your application and connecting it to the profiler. Here\textquotesingle{}s a quick refresher:\+


\begin{DoxyEnumerate}
\item Integrate Tracy Profiler into your application. This can be done using CMake, Meson, or simply by adding the source files to your project.
\item Make sure that {\ttfamily Tracy\+Client.\+cpp} (or the Tracy library) is included in your build.
\item Define {\ttfamily TRACY\+\_\+\+ENABLE} in your build configuration, for the whole application. Do not do it in a single source file because it won\textquotesingle{}t work.
\item Start your application, and {\itshape  Connect} to it with the profiler.
\end{DoxyEnumerate}

Please refer to the \href{https://github.com/wolfpld/tracy/releases}{\texttt{user manual}} for more details.
