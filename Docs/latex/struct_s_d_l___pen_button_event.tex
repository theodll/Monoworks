\doxysubsection{Detailed Description}
Pressure-\/sensitive pen button event structure (event.\+pbutton.\texorpdfstring{$\ast$}{*})

This is for buttons on the pen itself that the user might click. The pen itself pressing down to draw triggers a SDL\+\_\+\+EVENT\+\_\+\+PEN\+\_\+\+DOWN event instead.

\begin{DoxySince}{Since}
This struct is available since SDL 3.\+2.\+0.
