The easiest way to use SDL is to include it as a subproject in your project.

We\textquotesingle{}ll start by creating a simple project to build and run \href{hello.c}{\texttt{hello.\+c}}

First, you should have the Emscripten SDK installed from:\+

\href{https://emscripten.org/docs/getting_started/downloads.html}{\texttt{https:\+/\+/\+emscripten.\+org/\+docs/\+getting\+\_\+started/\+downloads.\+html}}

Create the file CMake\+Lists.\+txt 
\begin{DoxyCode}{0}
\DoxyCodeLine{cmake\_minimum\_required(VERSION\ 3.16)}
\DoxyCodeLine{project(hello)}
\DoxyCodeLine{}
\DoxyCodeLine{\#\ set\ the\ output\ directory\ for\ built\ objects.}
\DoxyCodeLine{\#\ This\ makes\ sure\ that\ the\ dynamic\ library\ goes\ into\ the\ build\ directory\ automatically.}
\DoxyCodeLine{set(CMAKE\_RUNTIME\_OUTPUT\_DIRECTORY\ "{}\$\{CMAKE\_BINARY\_DIR\}/\$<CONFIGURATION>"{})}
\DoxyCodeLine{set(CMAKE\_LIBRARY\_OUTPUT\_DIRECTORY\ "{}\$\{CMAKE\_BINARY\_DIR\}/\$<CONFIGURATION>"{})}
\DoxyCodeLine{}
\DoxyCodeLine{\#\ This\ assumes\ the\ SDL\ source\ is\ available\ in\ vendored/SDL}
\DoxyCodeLine{add\_subdirectory(vendored/SDL\ EXCLUDE\_FROM\_ALL)}
\DoxyCodeLine{}
\DoxyCodeLine{\#\ on\ Web\ targets,\ we\ need\ CMake\ to\ generate\ a\ HTML\ webpage.}
\DoxyCodeLine{if(EMSCRIPTEN)}
\DoxyCodeLine{\ \ set(CMAKE\_EXECUTABLE\_SUFFIX\ "{}.html"{}\ CACHE\ INTERNAL\ "{}"{})}
\DoxyCodeLine{endif()}
\DoxyCodeLine{}
\DoxyCodeLine{\#\ Create\ your\ game\ executable\ target\ as\ usual}
\DoxyCodeLine{add\_executable(hello\ WIN32\ hello.c)}
\DoxyCodeLine{}
\DoxyCodeLine{\#\ Link\ to\ the\ actual\ SDL3\ library.}
\DoxyCodeLine{target\_link\_libraries(hello\ PRIVATE\ SDL3::SDL3)}

\end{DoxyCode}


Build:\+ 
\begin{DoxyCode}{0}
\DoxyCodeLine{emcmake\ cmake\ -\/S\ .\ -\/B\ build}
\DoxyCodeLine{cd\ build}
\DoxyCodeLine{emmake\ make}

\end{DoxyCode}


You can now run your app by pointing a webserver at your build directory and connecting a web browser to it, opening hello.\+html

A more complete example is available at:\+

\href{https://github.com/Ravbug/sdl3-sample}{\texttt{https:\+/\+/\+github.\+com/\+\+Ravbug/\+sdl3-\/sample}}
