\doxysubsection{Detailed Description}
Touch finger event structure (event.\+tfinger.\texorpdfstring{$\ast$}{*})

Coordinates in this event are normalized. {\ttfamily \doxylink{struct_s_d_l___touch_finger_event_a0ce44b1342220fa17e9b9b4a77c2c906}{x}} and {\ttfamily \doxylink{struct_s_d_l___touch_finger_event_ac2bb8af638d2927a8e13f6ffe8f9384e}{y}} are normalized to a range between 0.\+0f and 1.\+0f, relative to the window, so (0,0) is the top left and (1,1) is the bottom right. Delta coordinates {\ttfamily \doxylink{struct_s_d_l___touch_finger_event_ac6acac209d6e2bd659fdb6760081393d}{dx}} and {\ttfamily \doxylink{struct_s_d_l___touch_finger_event_a9c0320c5f18a6b9d10da657e166608c9}{dy}} are normalized in the ranges of -\/1.\+0f (traversed all the way from the bottom or right to all the way up or left) to 1.\+0f (traversed all the way from the top or left to all the way down or right).

Note that while the coordinates are {\itshape normalized}, they are not {\itshape clamped}, which means in some circumstances you can get a value outside of this range. For example, a renderer using logical presentation might give a negative value when the touch is in the letterboxing. Some platforms might report a touch outside of the window, which will also be outside of the range.
