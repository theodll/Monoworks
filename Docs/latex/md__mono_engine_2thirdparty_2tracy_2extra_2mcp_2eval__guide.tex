\item GPU zones:\+ {\ttfamily get\+\_\+all\+\_\+gpu\+\_\+zone\+\_\+stats()}, {\ttfamily get\+\_\+gpu\+\_\+zone\+\_\+durations(...)}, {\ttfamily get\+\_\+gpu\+\_\+contexts()}

\item Capture metadata:\+ {\ttfamily get\+\_\+capture\+\_\+name()}, {\ttfamily get\+\_\+capture\+\_\+program()}, {\ttfamily get\+\_\+first\+\_\+time()}, {\ttfamily get\+\_\+last\+\_\+time()}, {\ttfamily get\+\_\+resolution()}, {\ttfamily get\+\_\+host\+\_\+info()}

\begin{DoxyItemize}
\item All time values returned by Worker methods are {\bfseries{nanoseconds}} (int). {\ttfamily get\+\_\+first\+\_\+time()} /\+ {\ttfamily get\+\_\+last\+\_\+time()} bound the capture timeline.
\item {\ttfamily Zone\+Stats} fields:\+ {\ttfamily count}, {\ttfamily total}, {\ttfamily min}, {\ttfamily max}, {\ttfamily avg}, {\ttfamily sum\+\_\+sq}. {\ttfamily total} is the inclusive aggregate; use {\ttfamily get\+\_\+child\+\_\+zone\+\_\+stats(srcloc\+\_\+id)} to subtract child time when you need self-\/time.
\item {\ttfamily get\+\_\+all\+\_\+zone\+\_\+stats()} returns {\ttfamily dict\+[str, Zone\+Stats]\+} keyed by an opaque label of the form {\ttfamily \textquotesingle{}name (addr)\+[arch]\+ \texorpdfstring{$<$}{<}srcloc\+\_\+id\texorpdfstring{$>$}{>}\textquotesingle{}}. The trailing {\ttfamily \texorpdfstring{$<$}{<}id\texorpdfstring{$>$}{>}} is the source-\/location ID — the int accepted by {\ttfamily get\+\_\+zone\+\_\+stats(int)}, {\ttfamily get\+\_\+zone\+\_\+durations\+\_\+by\+\_\+id}, and friends. Parse it with a regex if you need to join across calls.
\item Source-\/location IDs from {\ttfamily get\+\_\+all\+\_\+zone\+\_\+source\+\_\+locations()} are the join key between zone-\/name lookups and per-\/callsite queries.
