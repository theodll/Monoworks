Clicking on the {\itshape  Achievements} button opens the Achievements List. Here you can see the tasks to be completed along with a short description of what needs to be done.

As you complete each Achievement, new Achievements will appear, so be sure to keep checking the list for new ones!

To make the new things easier to spot, the Achievements List will show a marker next to them. The achievements {\itshape  Achievements} button will glow yellow when there are new things to see.
