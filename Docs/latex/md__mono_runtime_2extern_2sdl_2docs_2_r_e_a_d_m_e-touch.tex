Linux:\+ The linux touch system is currently based off event streams, and proc/\+bus/\+devices. The active user must be given permissions to read /\+dev/\+input/\+\+TOUCHDEVICE, where TOUCHDEVICE is the event stream for your device. Currently only Wacom tablets are supported. If you have an unsupported tablet contact me at \href{mailto:jim.tla+sdl_touch@gmail.com}{\texttt{jim.\+tla+sdl\+\_\+touch@gmail.\+com}} and I will help you get support for it.

Mac:\+ The Mac and i\+Phone APIs are pretty. If your touch device supports them then you\textquotesingle{}ll be fine. If it doesn\textquotesingle{}t, then there isn\textquotesingle{}t much we can do.

i\+Phone:\+ Works out of box.

SDL provides the ability to access the underlying \doxylink{struct_s_d_l___finger}{SDL\+\_\+\+Finger} structures. These structures should {\itshape never} be modified.

The following functions are included from \doxylink{_s_d_l__touch_8h_source}{SDL\+\_\+touch.\+h}

Devices are tracked by instance ID, of type SDL\+\_\+\+Touch\+ID.

To get a list of available device SDL\+\_\+\+Touch\+ID values, call SDL\+\_\+\+Get\+Touch\+Devices(). This returns an array of device IDs, terminated by a zero ID. Optionally, you can get a count of IDs by passing a non-\/\+NULL int\texorpdfstring{$\ast$}{*} to SDL\+\_\+\+Get\+Touch\+Devices() if you\textquotesingle{}d rather not iterate the whole array to get this number.

A SDL\+\_\+\+Touch\+ID may be used to get pointers to \doxylink{struct_s_d_l___finger}{SDL\+\_\+\+Finger}.

SDL\+\_\+\+Get\+Num\+Touch\+Fingers(touch\+ID) may be used to get the number of fingers currently down on the device.

The most common reason to access \doxylink{struct_s_d_l___finger}{SDL\+\_\+\+Finger} is to query the fingers outside the event. In most cases accessing the fingers is using the event. This would be accomplished by code like the following:\+ \begin{DoxyVerb}  float x = event.tfinger.x;
  float y = event.tfinger.y;
\end{DoxyVerb}


To get a \doxylink{struct_s_d_l___finger}{SDL\+\_\+\+Finger}, call SDL\+\_\+\+Get\+Touch\+Finger(\+SDL\+\_\+\+Touch\+ID touch\+ID, int index), where touch\+ID is a SDL\+\_\+\+Touch\+ID, and index is the requested finger. This returns a \doxylink{struct_s_d_l___finger}{SDL\+\_\+\+Finger} \texorpdfstring{$\ast$}{*}, or NULL if the finger does not exist, or has been removed. A \doxylink{struct_s_d_l___finger}{SDL\+\_\+\+Finger} is guaranteed to be persistent for the duration of a touch, but it will be de-\/allocated as soon as the finger is removed. This occurs when the SDL\+\_\+\+EVENT\+\_\+\+FINGER\+\_\+\+UP event is {\itshape added} to the event queue, and thus {\itshape before} the SDL\+\_\+\+EVENT\+\_\+\+FINGER\+\_\+\+UP event is polled. As a result, be very careful to check for NULL return values.

A \doxylink{struct_s_d_l___finger}{SDL\+\_\+\+Finger} has the following fields:\+
\begin{DoxyItemize}
\item x, y:\+ The current coordinates of the touch.
\item pressure:\+ The pressure of the touch.
\end{DoxyItemize}
