\doxysubsection{Detailed Description}
A rectangle stored using floating point values.

The origin of the coordinate space is in the top-\/left, with increasing values moving down and right. The properties {\ttfamily x} and {\ttfamily y} represent the coordinates of the top-\/left corner of the rectangle.
