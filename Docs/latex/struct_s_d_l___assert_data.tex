\doxysubsection{Detailed Description}
Information about an assertion failure.

This structure is filled in with information about a triggered assertion, used by the assertion handler, then added to the assertion report. This is returned as a linked list from SDL\+\_\+\+Get\+Assertion\+Report().

\begin{DoxySince}{Since}
This struct is available since SDL 3.\+2.\+0.
