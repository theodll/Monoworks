# Monoworks
Monoworks will be a complete set of game production software, designed to streamline core development pipelines and empower creators with world-class tools. The goal is to build a unified ecosystem that brings together asset management, engine tools and production workflows rather than relying on disparate utilities. Built from the ground up for performance and flexibility, it's designed for real world bottlenecks in modern game development.

Powered by modern APIs like Direct3D12 and Vulkan, and with modern multi threading, memory management, and concurrency design philosophies Monoworks will be performing exceptionally well and utillizing the full power of availlable hardware. Monoworks will also be easily extensible for future console or mobile support by splitting everything that is platform-specific into a platform-agnostic and platform-specific layer enabling easy extensibillity. Monoworks is architected like and to be like a professinonal game engine, consequently resulting in advanced features like automatic memory-defragmentation et cetera.



Somewhat up-to-date planning and development information can be found at https://sklout.theodll.me/monoworks.
